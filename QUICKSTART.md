# SalesIQ Quick Start Guide

## 🚀 Getting Started in 3 Steps

### Step 1: Access the Application
Open your browser and navigate to:
```
https://salesiq-000yl.app.super.betamyninja.ai
```

### Step 2: Upload Your First Sales Call
1. You'll land on the **Chat Assistant** page
2. Click the **"📎 Upload Sales Call Audio"** button
3. Select an MP3, WAV, M4A, or OGG file from your computer
4. Click **"Send"** to process the audio

### Step 3: View Your Report
1. Wait for the processing confirmation message
2. Navigate to the **"Reports"** tab in the left sidebar
3. Click **"View Details"** on your report
4. Review the comprehensive analysis

## 🎯 Common Tasks

### Analyzing a Sales Call
```
1. Go to Chat Assistant
2. Upload audio file (MP3, WAV, M4A, OGG)
3. Wait for processing (usually 30-60 seconds)
4. Check Reports section for results
```

### Viewing Report Details
```
1. Navigate to Reports tab
2. Find your report in the grid
3. Click "View Details" button
4. Review all sections:
   - Summary
   - Key Points
   - Sentiment Analysis
   - Action Items
   - Customer Needs
   - Full Transcription
```

### Managing Media Files
```
1. Go to Media Library tab
2. View all uploaded files
3. Check file size and upload date
4. Delete files you no longer need
```

### Deleting Reports
```
1. Go to Reports tab
2. Find the report to delete
3. Click "Delete" button
4. Confirm deletion
```

## 💡 Tips for Best Results

### Audio Quality
- ✅ Use clear, high-quality recordings
- ✅ Minimize background noise
- ✅ Ensure good microphone placement
- ❌ Avoid low-quality or distorted audio

### File Formats
- ✅ **MP3** - Best compatibility (recommended)
- ✅ **WAV** - High quality, larger files
- ✅ **M4A** - Good quality, Apple format
- ✅ **OGG** - Open format, good compression

### File Size
- ✅ Keep files under 25MB
- ✅ Compress large files if needed
- ✅ Split very long calls into segments
- ❌ Don't upload files over 25MB

## 🆘 Troubleshooting

### Upload Not Working
**Problem:** File won't upload
**Solutions:**
- Check file format (MP3, WAV, M4A, OGG only)
- Verify file size (must be under 25MB)
- Try a different browser
- Refresh the page and try again

### Report Not Appearing
**Problem:** Report doesn't show after upload
**Solutions:**
- Wait 30-60 seconds for processing
- Refresh the Reports page
- Check the Chat for error messages
- Verify the upload completed successfully

### Page Not Loading
**Problem:** Application won't load
**Solutions:**
- Check your internet connection
- Clear browser cache
- Try a different browser
- Verify the URL is correct

## 🎓 Example Workflow

### Complete Sales Call Analysis

**Scenario:** You just finished a 30-minute sales call with a potential customer.

**Steps:**
1. **Record the call** (with permission)
2. **Save as MP3** on your computer
